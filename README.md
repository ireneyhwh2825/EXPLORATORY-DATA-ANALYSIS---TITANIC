# EXPLORATORY-DATA-ANALYSIS---TITANIC
Performed Exploratory Data Analysis on the Titanic dataset, including data cleaning, visualization, correlation analysis, and insight generation.

## Overview
Exploratory Data Analysis of the Titanic dataset using Python.

## Features
- Missing value handling
- Duplicate value analysis
- Data visualization
- Correlation heatmap
- Insight generation
